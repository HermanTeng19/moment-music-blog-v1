# 音频文件目录

请将您的音频文件放置在此目录中。

## 支持的格式
- MP3 (.mp3) - 推荐
- WAV (.wav)
- OGG (.ogg)
- M4A (.m4a)

## 文件命名规范
建议使用以下命名格式：
- `song-001.mp3`
- `song-002.mp3`
- `song-003.mp3`

## 示例文件
为了演示播放器功能，您需要添加以下音频文件：
1. `song-001.mp3` - 对应"第061期 Wild"
2. `song-002.mp3` - 对应"夏日微风"
3. `song-003.mp3` - 对应"城市之夜"

## 注意事项
- 确保音频文件路径与JSON元数据中的`audioSrc`字段一致
- 推荐使用相对路径引用
- 音频文件大小建议控制在合理范围内以确保加载速度
